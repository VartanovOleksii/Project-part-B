## Мета роботи

* Навчитися будувати ієрархії класів, у тому числі успадковуючись від абстрактного класу, a також використовувати концепцію поліморфізму в ООП.

## Завдання

1.  Удосконалити проект, скорегував діаграму класів, додавши реалізацію хоча б одним класом будь-якого інтерфейсу(ів) .NET (IClonable, IComparable, IEnumereable тощо).
2.  Передбачити множинну реалізацію інтерфейсів.
3.  Додати абстрактний клас, від якого успадкувати декілька нових чи вже існуючих класів.
4.  Передбачити різні варіанти реалізації поліморфізму: перевизначення віртуального методу базового класу в похідних класах, динамічне визначення типу об'єкта і виклик відповідної реалізації методу;  використання змінних типу інтерфейс, динамічне визначення типу об'єкта і виклик відповідної реалізації методу.
5.  Скорегувати каркас проєкту.
6.  Замінити «заглушки» в проєкті конкретними реалізаціями методів, властивостей тощо.
7.  Запустити unit-тести, досягти проходження всіх unit-тестів.
8.  Спроектувати меню додатка, пункти меню мають відповідати предметній області й бути логічними.
9.  Реалізувати основну програму (файл Program.cs) відповідно до спроектованого меню.
10.  Виконати функціональне тестування всіх пунктів меню.
11.  Оформити звіт.

## Опис Класів та Поведінка

### Клас `Artist`

* **Основні дані:** Ім'я, Вік, Зарплатня, Інструмент, Активність, Кількість фанатів.
* **Валідація:** Ім'я (3-50 символів), Вік (18-95), Зарплатня (> 0), Інструмент (3-50, латинські символи/цифри), Кількість фанатів ($\ge 0$).

### Клас `Band`

* **Основні дані:** Назва, Кількість прослуховувань за місяць, Продюсери, Артисти.
* **Валідація:** Назва (3-50 символів), Кількість прослуховувань за місяць ($\ge 0$).

### Клас `Producer`

* **Основні дані:** Ім'я, Вік, Зарплатня, Спеціалізація, Стаж.
* **Валідація:** Ім'я (3-50 символів), Вік (20-95), Зарплатня (> 0), Спеціалізація (3-50 символів), Стаж (3-90).

### Клас `Person`

* **Основні дані:** Ім'я, Вік, Зарплатня.

### Клас `RecordLabel`

* **Основні дані:** Назва (3-50 символів), Групи.
* **Поведінка:** Додати групу, Відписати групу.

### Клас `Song`

* **Основні дані:** Назва (3-50 символів), Кількість прослуховувань ($\ge 0$), Група, Жанр.
* **Поведінка:** Запустити, Зупинити, Вивести метадані.
